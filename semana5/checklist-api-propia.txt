¿Las rutas entregan datos estructurados?

Respuesta: SÍ.
Explicación: Las respuestas del backend viajan en formato JSON bien

8. Render dinámico
¿El tablero muestra datos recibidos desde la API?

Respuesta: SÍ.
Explicación: Tras pulsar las funciones de carga, los contenedores vacíos
procesan el JSON recibido y pintan dinámicamente las tarjetas del tablero
y las filas correspondientes al calendario editorial.


9. Diagnóstico básico
¿Sabes qué revisar si no carga la información?

Respuesta: SÍ.
Explicación: Se debe verificar que el servidor local en Node.js esté
corriendo en el puerto 3000, revisar la pestaña Network/Red en las
Herramientas de Desarrollador del navegador para validar el estado de las
peticiones HTTP (200 OK) y confirmar la correcta lectura de los archivos JSON.

¿Los datos dinámicos mantienen claridad, fuente y lenguaje adecuado?

Respuesta: Si.
Explicación: La información servida de forma dinámica mantiene un tono
